# ¡Hola!
Este proyecto consiste en el diseño y creación de un CV web realizado en HTML para la materia Computación III de la Lic. en Diseño y Comunicación Visual.
La consigna era crear una página web personal para experimentar con las etiquetas HTML, aplicar estilos con CSS y organizar el contenido de la página con Flexbox. 
Además trabajé con tipografías externas de Google Fonts e imágenes SVG para mantener la buena resolución de los vectores.

## Creación
La idea fue empezar a construir mi CV en formato digital, y buscar una presentación más visual y diseñada, y menos tradicional, como una aproximación a mi primer portfolio.
Gracias a este trabajo estuve muchas noches viendo letritas y brackets en la pantalla pero creo que quedó bien y a mi gusto. Hice lo que pude con lo que tenía.
Empecé con un HTML sin alma y CSS me lo convirtió en una princesa.

### Detalles técnicos
* Mi pantalla tiene una resolución de 1366x768, pero la página fue diseñada en un formato de resolución 1920x1080, así que los contenidos se adaptan a ambas y al celular.
* El editor de código que usé es Visual Studio Code.
